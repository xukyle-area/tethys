# **Java 位运算符**

## **基本位运算**
```java
// 1. 与运算 (AND) - &
int a = 5;  // 二进制: 101
int b = 3;  // 二进制: 011
int result = a & b;  // 结果: 001 (十进制: 1)

// 2. 或运算 (OR) - |
result = a | b;  // 结果: 111 (十进制: 7)

// 3. 异或运算 (XOR) - ^
result = a ^ b;  // 结果: 110 (十进制: 6)

// 4. 非运算 (NOT) - ~
result = ~a;  // 结果: ...11111010 (按位取反)
```

## **移位运算**
```java
int num = 8;  // 二进制: 1000

// 左移 (LEFT SHIFT) - <<
int leftShift = num << 2;   // 结果: 32 (1000 -> 100000)

// 右移 (RIGHT SHIFT) - >>
int rightShift = num >> 2;  // 结果: 2 (1000 -> 10)

// 无符号右移 - >>>
int unsignedRight = num >>> 2;  // 结果: 2
```

## **常用位运算技巧**

### **1. 判断奇偶**
```java
boolean isOdd = (n & 1) == 1;  // 奇数
boolean isEven = (n & 1) == 0; // 偶数
```

### **2. 交换两数 (不用临时变量)**
```java
a = a ^ b;
b = a ^ b;  // b = (a^b)^b = a
a = a ^ b;  // a = (a^b)^a = b
```

### **3. 获取/设置特定位**
```java
// 获取第k位 (从右数第k位，从0开始)
boolean getBit = (n & (1 << k)) != 0;

// 设置第k位为1
n = n | (1 << k);

// 清除第k位 (设为0)
n = n & ~(1 << k);

// 切换第k位
n = n ^ (1 << k);
```

### **4. 计算二进制中1的个数**
```java
// 方法1: Brian Kernighan算法
int countOnes(int n) {
    int count = 0;
    while (n != 0) {
        n &= (n - 1);  // 每次消除最右边的1
        count++;
    }
    return count;
}

// 方法2: 内置方法
int count = Integer.bitCount(n);
```

### **5. 检查是否为2的幂**
```java
boolean isPowerOfTwo = n > 0 && (n & (n - 1)) == 0;
```

## **LeetCode 常见位运算题目**
```java
// 136. Single Number - 找出数组中只出现一次的数字
public int singleNumber(int[] nums) {
    int result = 0;
    for (int num : nums) {
        result ^= num;  // 相同数字异或为0，0异或任何数为该数本身
    }
    return result;
}

// 191. Number of 1 Bits - 计算1的个数
public int hammingWeight(int n) {
    int count = 0;
    while (n != 0) {
        count++;
        n &= (n - 1);  // 消除最右边的1
    }
    return count;
}

// 190. Reverse Bits - 翻转二进制位
public int reverseBits(int n) {
    int result = 0;
    for (int i = 0; i < 32; i++) {
        result <<= 1;           // 左移一位
        result |= (n & 1);      // 取n的最低位加到result
        n >>= 1;                // n右移一位
    }
    return result;
}
```

## **优先级注意事项**
```java
// 位运算优先级较低，注意加括号
if ((x & mask) == 0) { ... }  // 正确
if (x & mask == 0) { ... }    // 错误! 等价于 x & (mask == 0)
```
