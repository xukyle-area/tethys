## Basic DFS/BFS

### 104. [Maximum Depth](https://leetcode.com/problems/maximum-depth-of-binary-tree/)
```java
class Solution {
    public int maxDepth(TreeNode node) {
        if (node == null) return 0;
        return Math.max(maxDepth(node.right), maxDepth(node.left)) + 1;
    }
}
```
